# wow-addon

## location
would go into
C:\[...]\World of Warcraft\_retail_\Interface\AddOns  

## toc
comment out file with #  
`Mounts.lua`  
becomes  
`#Mounts.lua`  

## slash commands
`/auau`
* show if tracking

`/auau off`
* disable tracking

`/auau on`
* enable tracking

## addon options
`/auui`
* opens settings
